1. Que son los arrays , definicion ejemplo de cada tipo de dato (incluyendo indices)

Es una colección o agrupación de elementos en una misma variable, cada uno de ellos ubicado por la posición que ocupa en el array

Ejemplos:

const arrays = {'Marco',28,true,1.81}

console.log(arrays[0],arrays[1],arrays[2],arrays[3])

Devuelve: Marco 28 true 1.81

2. Que tipos de datos podemos agregar en los arrays(con ejemplos)

Se puede agregar diferentes tipos de datos en un arreglo, o simplemente de un sólo tipo. JavaScript nos dá esa flexibilidad.

Ejemplos:

Arrays de Strings => const nombres = ['Marco','Luis','Roberto']

Arrays de Numbers => const edades = [25,26,48]

Arrays de Varios => const varios = ['Marco',25,true]

Arrays de Objetos => const persona = [{dni:'45856869',nombre:'Marco'},{dni:'65898596',nombre:'Luis'}]

3. Explicar los metodos o metodo enseniado en clase para los arreglos(definicion y ejemplo)

Ejemplo:

length: este método devuelve la cantidad de elementos de un arreglo.

const muebles = ['sillas','ropero','mesa','cuadro'];

muebles.length //Cantidad de elementos: 4

4. Como acceder a cada elemento del arreglo(definicion y ejemplo)

Se puede acceder por la posición del elemento.

Ejemplo:

const muebles = ['sillas','ropero','mesa','cuadro'];

console.log(muebles[2]); //Devuelve: 'mesa'

5. Que son los arreglos bidimensionales( definicion y ejemplo)

Los arreglos bidimensionales, es cuando un arreglo se encuentra dentro de otro arreglo.

Ejemplo:

const productos = ['leche','pescado','huevo',['sal','pimienta','comino']]

6. Como iterar los arreglos , de manera imperativa(for) y de manera declarativa (map y forEach)

Ejemplo:

Manera Imperativa: FOR

for(let i=0, i < arreglo.lenght, i++) { }

Manera Declarativa: MAP y FOREACH

muebles.foreach(function(mueble){ console.log(mueble); });

muebles.map(function(mueble){ console.log(mueble); });

7. Diferencias con ejemplos de map y forEach

Ejemplos:

FOREACH

Recorre un arreglo

muebles.foreach(function(mueble){ console.log(mueble); });

MAP

Recorre un arreglo, pero este se puede guardar en una variable, generando otro arreglo.

const misMuebles = muebles.foreach(function(mueble){ return mueble.toUpperCase(); });
